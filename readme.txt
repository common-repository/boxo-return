== Description ==

BOXO Return is een universeel retoursysteem voor herbruikbare verzendverpakkingen in Nederland. 
Met deze app bied je klanten de optie voor een herbruikbare verzendverpakking in de checkout. Wanneer je klant hiervoor kiest wordt statiegeld toegevoegd in de winkelmand. Deze ontvangt de klant direct terug na inlevering van de verpakking bij een van de inleverpunten door heel Nederland.
